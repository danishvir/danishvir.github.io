Computer History in the 2000s

What key events occured in computer history during the 2000s?

1.The first 64-bit processor becomes available to the consumer market in 2003.

The first 64-bit processor became available to the consumer market in 2003.It was the Athlon 64.It was created by Advanced Micro Devices (AMD).

2. Microsoft launches Windows 7.

In 2009, Microsoft launched Windows 7. Windows 7 had some new features. These new features included the ability to pin applications to the taskbar, among others

3.The iPhone, in 2007, brought many computer functions to the cell phone.

The iPhone brings many computer functions to the iPhone cell phone in 2007. There are many new features that were previously not present in the iPhone.

4. Apple introduces the MacBook Pro, its first Intel-based dual-core mobile computer.

In 2006,Apple introduced acBook Pro. The MacBook Pro was Apple's first ever Intel-based, dual-core mobile computer.

5. YouTube is founded.

In 2005, YouTube is founded by Jawed Karim, Chad Hurley, and Steve Chen, all former PayPal employees. Google bought YouTube on November 13,2006.
